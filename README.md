# 🌍 Tourism Map PWA

A modern Tourism Map application built with **Next.js**, **TypeScript**, **Leaflet**, and **OpenStreetMap**.

The project demonstrates how to build an interactive map experience with hotel locations, search functionality, responsive UI, and Progressive Web App (PWA) support.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/tourism-map-pwa.git
```

Install dependencies

```bash
pnpm install
```

Run development server

```bash
pnpm dev
```

Production build

```bash
pnpm build
pnpm start
```
